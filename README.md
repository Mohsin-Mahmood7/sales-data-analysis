# Sales Data Analysis Project

## Overview
This project analyzes sales data using Excel to identify trends, regional performance, and category insights.

## What I Did
- Cleaned and prepared raw sales data
- Created pivot tables for analysis
- Built a dashboard to visualize key metrics

## Dashboard Insights
- Highest daily sales occurred on May 25, 2024 totaling $112,000.
- Sales peaked in May compared to previous months.
- North and South regions showed strong performance.

## Tools Used
- Microsoft Excel (Pivot Tables, Charts, Dashboard)
